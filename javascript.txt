1.Introduction 
- Javascript is web based scripting Language.
- web scripting Language type :- 1) Client-side scripting 2)Server-side scripting 
- Javascript is a Event Based Programming Language.
- Example Event  :- Click,Right Click, Mouse hover, Key press,load,Resize ..etc....etc
- Js is used to create client-side dynamic Pages.
- Javascript is a case-sensitive Language.

Fetures  
- All popular web Browsers support Javascript.
- Javascript is an Object-orinted Programming Language.
- Javascript follows the syntax and structure of the C Programming Language. 
- Javascript support Operating System including,windows,macOs,Linux etc.
